Our story

Two friends, one love of cruiser bikes.

Lifelong friends Adam and Penelope turned a shared love of cruiser bikes into a business built on one idea: riding should be easy, comfortable, and fun. What started as a single storefront has grown into a name riders across British Columbia recognize.

2017

Ace Bikes opens its first location in Lonsdale, North Vancouver — an instant hit with local cruiser riders.

2020

Demand grows through the pandemic, and Ace Bikes launches its e-commerce shop to keep every rider outfitted.

Today

Locations across the province, from Victoria to Kelowna, all outfitting the leisure bike enthusiast.

This is where the storefront ends and the coursework begins. Everything below connects to the same live Ace Bikes data, structured for Power Query.

1. Connect in Power Query (no code needed)

For each table you want, repeat these steps:

  1. Data tab → Get Data → From Web
  2. Paste the table's URL (copy one from the card on the right) → OK
  3. Click To Table → OK
  4. Click the expand icon on the column header → Expand → OK

2. Copy a table's URL

Each URL already includes the course access key — just copy and paste.
