Connecting with Power Query
  1. 1
    In Excel, go to Data → Get Data → From Other Sources → From Web
  2. 2
    Enter the table endpoint URL:
    [Base URL]/rest/v1/[table_name]
  3. 3
    In the HTTP request headers, add:
    apikey: [your API key]
  4. 4
    Set Authorization header to:
    Bearer [your API key]
  5. 5
    Load the JSON response into Power Query and transform as needed
  6. 6
    Repeat for each table, then load all tables to the Data Model
